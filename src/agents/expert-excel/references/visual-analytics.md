# Visual Analytics Mastery (Chart Design)

## 📊 Core Principles
1.  **One Chart, One Insight**: Every chart must have a narrative takeaway as its title. Do not use generic titles like "Sales Data".
2.  **Strategic Chart Selection**:
    -   **Trends over time**: Line charts (continuous data).
    -   **Comparisons**: Bar/Column charts (discrete categories).
    -   **Drill-down / Variances**: Waterfall or Box-plots (outliers and breakdown).
3.  **Aesthetics**: 
    -   Maintain high contrast.
    -   Use Bento Grid dashboard layout when generating multiple charts.
    -   Minimize gridlines and "chart junk" to focus attention on the data.
4.  **Dynamic Interactivity**: Utilize Slicers and PivotCharts for real-time user-driven exploration when building dashboards.
